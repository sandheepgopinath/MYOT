# **App Name**: MYOT: Make Your Own Tee

## Core Features:

- T-Shirt Selection: Allow users to select a T-shirt based on GSM (100, 200, 220), fit (oversized, normal), and view the base price.
- Design Selection: Users can select from a collection of pre-made designs (Zodiac, Thrissur, etc.). Each design has a price.
- Placement Options: Offer various placement options for the selected design (front, back top, back center, back bottom, front bottom, etc.).
- Custom Uploads: Allow users to upload their own designs and/or add custom text.
- Design Review: Use AI tool to review submitted designs for obscenity, religious/hateful content, and potential copyright infringement before order placement.
- Dynamic Pricing: Calculate and display the final price based on the T-shirt type, selected design(s), and any custom uploads.
- Design Submission for Mass Production: Enable users to submit uploaded designs for mass production consideration, offering a percentage of profit for each print.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082), evoking creativity and sophistication related to unique design.
- Background color: Very light lavender (#F0E6EF), a close but heavily desaturated variant of the primary color, maintaining the sophistication with subtlety.
- Accent color: Violet (#8F00FF), analogous to indigo, and distinct in both brightness and saturation to highlight key elements and calls to action.
- Headline font: 'Belleza', sans-serif. For the body of the page and descriptions of your shirts and policies, use 'Alegreya', serif.
- Simple, modern icons representing various design categories, placement options, and GSM values.
- Clean, grid-based layout with a focus on showcasing T-shirt designs and customization options.
- Subtle transitions and animations on button hovers and when adding/removing designs.
